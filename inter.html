What is RxJS?

1.A utility for handling asynchronous data streams.

2.Like a Promise that can resolve over and over again.

RxJS terminology

Observable - An array that is built over time.

Subscription - A for loop that happens over time.

Emit - When an Observable spits out a value because something subscribed to it.

Operator - Functions that alter the the way Observables emit values.

What is the difference between an observable and a subject?

Probably a more important distinction between Subject and Observable is that a Subject has state, it keeps a list of observers. On the other hand, an Observable is really just a function that sets up observation.

When do you use template driven vs model driven forms? Why?

Template-driven forms are asynchronous in nature, whereas Reactive forms are mostly synchronous. In a template-driven approach, most of the logic is driven from the template, whereas in reactive-driven approach, the logic resides mainly in the component or typescript code

The prototype is an object that is associated with every functions and objects by default in JavaScript.
